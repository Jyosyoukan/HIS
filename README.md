# HIS
《程序设计基础课程设计》 （2025级） 
编写一个c语言程序，实现一个适用于小型医院的轻量级的医疗管理系统（HIS） ，能够管理至少100名普通患者 ，30名住院患者 ，20名医生 ，5个科室，三种病房 ，30类药品， 其中：
1.	登录窗口
2.	医生登录进去 看到自己信息  看到出诊时间  按日期看
3.	输入123进入对菜单
4.	给病人修改病例  给病人开药  
5.药房的工作人员 登录进去 看到自己信息 给个搜索框 搜药 改药  出库 进库  药品太少提示 药品过期提示
6.病人缴费药品自动减少
7.登录  进去  用户名密码    
8.当日  预约挂号  查看已经挂的号  账单  （）当日在后 
9. 当日 进入   选科室  
10.选医生 1 2 3 4 时间可选 不可选有显示   
11.缴费
